🧠 AI Dataset Visual Analyzer
A Streamlit web app that transforms any CSV dataset into infographic-quality visual analysis — built for AI training pipeline workflows.

What it does
Dataset Overview — rows, columns, feature types at a glance
Data Quality Score — automated scoring across completeness, uniqueness, and outlier health
Missing Values Heatmap — per-column missing data visualization
Feature Distributions — histogram grid for all numeric features
Correlation Matrix — seaborn heatmap for feature relationships
Label Distribution — donut chart for classification targets
Run Locally
pip install -r requirements.txt
streamlit run app.py
Tech Stack
Python · Streamlit · Pandas · Matplotlib · Seaborn · NumPy
